# Problem Definition

## Description

Given n points on a 2D plane, find the maximum number of points that lie on the same straight line.

Sample Input :

```text
(1, 1)
(2, 2)
```

Sample Output: 2

You will be give 2 arrays X and Y. Each point is represented by (X[i], Y[i])
