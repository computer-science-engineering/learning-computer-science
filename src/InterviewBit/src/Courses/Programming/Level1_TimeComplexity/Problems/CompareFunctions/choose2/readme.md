# Problem Definition

## Description

Which of the given options provides the increasing order of complexity of functions f1, f2, f3 and f4:

```plaintext
f1(n) = 2^n

f2(n) = n^(3/2)

f3(n) = nLogn

f4(n) = n^(Logn)
```

### Choices

* f3, f2, f4, f1
* f3, f2, f1, f4
* f2, f3, f1, f4
* f2, f3, f4, f1

### Answer

f3, f2, f4, f1

Look at the values for functions for very large value of n.
