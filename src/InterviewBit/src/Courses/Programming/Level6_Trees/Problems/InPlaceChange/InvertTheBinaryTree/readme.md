# Problem Definition

## Description

Given a binary tree, invert the binary tree and return it. 
Look at the example for more details.

Example:

Given binary tree

```text
     1
   /   \
  2     3
 / \   / \
4   5 6   7
```

invert and return

```text
     1
   /   \
  3     2
 / \   / \
7   6 5   4
```
