# Problem Definition

## Description

Given a list of appointments, find all the conflicting appointments.

Example:

```text
Appointments: [[4,5], [2,3], [3,6], [5,7], [7,8]]
Output:
[4,5] and [3,6] conflict.
[3,6] and [5,7] conflict.
```
